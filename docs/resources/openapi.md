# OpenAPI

The public Merchant API contract is generated from the backend endpoint metadata and DTOs.

- [OpenAPI JSON](https://api.stendly.com/openapi.json)
- [OpenAPI YAML](https://api.stendly.com/openapi.yaml)

The document covers mainnet and devnet servers, bearer API-key authentication, payment intents, merchant profile and
statistics, terminals, webhook configuration, standard error schemas, idempotency, and outgoing webhook callbacks.
